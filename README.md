# 随机文件夹浏览器 (RandomFolderBrowser)

一款轻量级、高效率的 Windows 本地目录随机探索工具。专为海量图片、视频等多媒体资源的管理与随机浏览设计，让每一次开启都充满惊喜。

> **当前版本：** v1.06  
> **开发框架：** .NET Windows Forms (WinForms)  
> **主题风格：** 沉浸式暗色主题

---

## 🚀 核心功能亮点

- **🧠 智能缓存系统**  
  基于 `Directory.EnumerateFiles` 快速扫描并建立文件夹索引。支持**父子文件夹双向缓存**：子文件夹可自动读取父文件夹缓存，扫描子目录后也会自动合并回父缓存，效率极高。

- **🎲 完全公平轮询（开关可选）**  
  摒弃传统概率随机，内置**轮流随机机制**（开关可配置）。开启后，所有可用文件夹将被放入池中轮询，确保没有重复踩坑，真正做到“雨露均沾”。

- **🏷️ 标签过滤系统**  
  为特定文件夹打上分类标签（如“风景”、“摄影”、“素材”）。随机时可通过多选标签精准过滤范围。标签数据保存在 `setting/tags.dat` 中，跨会话保留。

- **🖱️ 鼠标中键快捷操作**  
  在软件任意空白区域点击鼠标中键，即可触发一键随机。可在设置中自由配置为“随机打开文件夹”或“随机打开文件”。

- **🎞️ 滚轮连看（无缝切换）**  
  随机打开一张图片或一段视频后，直接在软件界面空白区**向上滚动滚轮（上一张/上一个视频）**，**向下滚动滚轮（下一张/下一个视频）**，无需频繁点击，浏览体验极其流畅。

- **🌙 极致暗色视觉体验**  
  采用精心调配的低饱和度黑灰色调（`#1E1E1E` 与 `#2D2D30`），减少视觉疲劳，在夜间或暗光环境下尤其舒适。

- **📦 内置自动更新**  
  无需手动去官网下载。软件内置自动更新功能，启动时比对 GitHub 根目录下的 `version.txt`。更新过程使用 `update.bat` 脚本自动替换并重启（**已完美支持中文路径下的更新**）。

## 📖 使用指南

1. **设置基础路径**：在界面顶部输入需要浏览的根目录路径，点击“浏览”按钮选择。
2. **建立缓存**：点击右侧的 **【扫描】** 按钮。软件会遍历所有子文件夹并建立索引（第一次扫描可能稍慢，之后秒开）。
3. **开始随机**：
   - 点击 **【📁 随机打开文件夹】**：立刻打开一个随机文件夹。
   - 点击 **【📄 随机打开文件】**：从所有符合后缀过滤条件的文件中随机打开一个。
4. **设置过滤与个性化**：点击左上角标题 `⚙随机文件浏览器`，进入设置弹窗，可配置鼠标中键行为、查看器/播放器路径、文件后缀开关以及轮询机制。

## 📂 数据与配置目录说明

程序运行后在根目录下会自动生成以下文件夹：

| 文件夹 | 说明 |
| :--- | :--- |
| **`setting/`** | 存储用户设置 (`settings.ini`) 和标签数据 (`tags.dat`)。 |
| **`cache/`** | 存放扫描后的文件夹索引缓存文件，提升下次启动速度。 |
| **`reported/`** | 存放调试日志和性能监控记录（按天生成 `.log` 文件，自动保留最近 2 天）。 |

---

## 📦 自动更新注意事项（开发者/打包者须知）

更新机制基于 GitHub Raw 读取，务必保证仓库 `main` 分支根目录下存在以下 **2 个文件**：

1. `version.txt`（纯文本，仅包含版本号，如 `1.06`）
2. `随机文件夹.exe`（编译好的、无版本号后缀的可执行程序）

> **提示**：自动更新脚本通过 `chcp 65001` 强制切换编码，已完美兼容中文目录路径，无需用户额外调整。
